Brief Explanations for the Web Site Resources Listed

learn.shayhowe.com/html-css/
A comprehensive guide to learn the fundamentals of HTML.

codeavengers.com/
Provides interactive online tutorials and uses games and apps to help users learn HTML.

activejump.com/
Is a smart-wiki tutorial page for HTML.

channel9.msdn.com/
Is a video blog created by Microsoft employees to provide demonstrations on the latest trends in technology. It also provides resources on the fundamentals of HTML.

htmldog.com/guides/html/beginner/
Has step-by-step tutorials and guides for HTML broken up into beginner, intermediate, and advanced categories.

simplehtmlguide.com/cheatsheet.php
Is a cheat sheet which lists HTML tag definitions.

html-5-tutorial.com/
Covers the basics of HTML and provides necessary skills to develop basic webpages.

code.tutsplus.com/categories/html
Has tutorials, articles, and downloadable project files to practice with.

docs.webplatform.org/wiki/Beginners
Has articles, tutorials, and information on various text editors.

stackoverflow.com/questions/tagged/html5
Is a programming forum you can use to ask questions and receive feedback from other developers. There is also a mini-site just for HTML coding.
